# Zentao Log Agent 1.0.1

本版本修复 Windows 目录选择和首次启动默认配置问题。

## 修复内容

- Windows 目录选择器显示所有可访问盘符，可从 `C:` 切换到 `D:` 等其他磁盘
- 禅道登录地址默认使用 `http://192.168.0.216:30085/user-login.html`
- 禅道任务页地址默认使用 `http://192.168.0.216:30085/my-work-task.html`
- 已有配置中的地址为空时自动补齐默认值，自定义地址不会被覆盖

## 安装说明

本版本暂未配置 Apple 和 Windows 代码签名证书。首次启动时，操作系统可能显示来源或安全提醒。
