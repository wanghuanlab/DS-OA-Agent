# Zentao Log Agent 1.0.3

本版本新增云之家考勤工时自动统计。客户端可以读取所选日期的个人打卡流水，用最早与最晚有效打卡计算每日工时，并把结果带入日志预览。

## 核心亮点

- **自动读取考勤工时**：通过已登录的云之家个人考勤 URL 获取打卡流水，按每天最早和最晚有效打卡计算工时
- **多任务自动分配**：同一天有多个禅道任务时，以 0.5 小时为粒度分配当天工时，并保持当日总工时一致
- **可选且不阻断**：未登录、凭证过期、网络超时或没有完整打卡记录时只提示，不影响日志生成、预览保存和禅道提交
- **本地优先**：云之家临时授权 URL 与其他配置一样，仅保存在当前电脑的本地 JSON

## 更新内容

- 设置页新增“云之家考勤”URL 配置、自动保存与“检测考勤”按钮
- 预览日期下方展示当天最早打卡、最晚打卡和计算工时
- 生成预览时自动读取所选周期的个人考勤记录
- 新增临时凭证缺失、过期、请求超时和记录不完整的明确反馈
- 官网新增操作手册，说明云之家授权 URL 的获取和检测步骤
- 默认配置只包含不带 `ticket` 的云之家页面地址，不内置任何用户凭证

## 安装说明

本版本暂未配置 Apple 和 Windows 代码签名证书。首次启动时，操作系统可能显示来源或安全提醒。
