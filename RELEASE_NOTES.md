# Zentao Log Agent 1.0.0

首个正式桌面版本，面向需要从代码提交整理并录入禅道工作日志的开发者。

## 主要能力

- 支持 Git、Mercurial 和 SVN 代码库自动识别与提交汇总
- 每个代码库可关联禅道任务、提交人与附加描述
- 同一日期可添加多条任务、工作描述和耗时
- 提交时根据任务预计与消耗信息续算预计剩余工时
- 禅道、LLM、代码库和任务关联配置自动保存到本机 JSON
- 每周五 16:00 自动生成预览，17:00 自动录入
- 提供 macOS Apple Silicon、macOS Intel 和 Windows 客户端

## 安装说明

本版本暂未配置 Apple 和 Windows 代码签名证书。首次启动时，操作系统可能显示来源或安全提醒。
