# Zentao Log Agent 1.0.2

本版本带来全新的桌面工作台与正式应用品牌，让“读取代码提交、确认工作总结、直接录入禅道”成为一条清晰、可控的本地流程。

## 核心亮点

- **自动总结代码工作**：自动识别 Git、HG、SVN，按日期与提交人读取记录，并生成中文工作描述
- **VPN 连接后直接填报**：无需打开禅道网页，即可检测状态、获取任务并通过 HTTP 接口提交日志
- **本地优先且透明**：源代码公开可审查，禅道、LLM、代码库和任务配置只保存在当前用户的本地 JSON

## 更新内容

- 新增“准备工作、确认预览、录入禅道”三步工作台
- 禅道与 LLM 配置移动到独立设置页面，并保持自动保存
- 新增正式应用 LOGO，覆盖 macOS、Windows、托盘、应用界面与官网
- 移除代码库的“附加描述”输入及“附加描述汇总”区域
- 移除定时生成预览与自动录入功能，所有填报均由用户确认后手动执行
- 代码库配置简化为目录、关联任务和提交人
- 同一日期支持添加多条任务、工作描述和耗时
- 根据任务初始预计、累计消耗与本次耗时计算预计剩余
- 官网更新产品介绍与新版界面截图

## 安装说明

本版本暂未配置 Apple 和 Windows 代码签名证书。首次启动时，操作系统可能显示来源或安全提醒。
