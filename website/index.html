为开发者准备的本地工作日志助手

Zentao Log Agent

把 Git、HG、SVN 提交与附加描述整理成中文工作日志。你负责确认，它负责按时录入禅道。

选择下载版本 查看使用方式
本地运行 提交记录总结 可编辑预览

少做重复录入

工作已经做过一次，不必再回忆一次

Agent 在你的电脑上读取版本库提交，结合每个代码库的附加描述和任务关联，生成按日期拆分的日志预览。提交前，每一天都能继续增删任务、调整描述与耗时。

01

多版本库，自动识别

选择目录即可检查 Git、HG 或 SVN，无需先指定类型。普通目录会被安静忽略。

GitMercurialSubversion

02

每个代码库关联任务

保留上次选择的提交人与任务映射，多项目并行时无需反复配置。

03

剩余工时自动续算

录入时根据任务初始预计、累计消耗和本次耗时计算预计剩余。

04

配置留在本机

禅道和 LLM 设置保存在当前用户的数据目录，项目仓库不收录个人配置文件。

提交前看清每一条

同一天，可以有多个任务

日期、任务、工作描述和耗时都能直接编辑。任务剩余时间同时展示，工作拆分更直观。

三步完成一周日志

从本地提交到禅道

  1. 连接填写禅道和 LLM 配置，选择本机代码库与提交人。
  2. 确认生成中文预览，按日期补充任务、描述和实际耗时。
  3. 录入立即提交，或让客户端在周五 16:00 生成预览、17:00 自动执行。

在自己的电脑上运行

选择你的平台

macOS 支持 Apple Silicon 与 Intel，Windows 提供安装版。首次运行后，配置会自动保存在当前用户目录。
